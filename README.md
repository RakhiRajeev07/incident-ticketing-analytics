# Incident Ticketing Analytics
his project performs end-to-end data analysis of IT service incident tickets using Python, SQL, and Power BI.  
The goal is to support business operations, improve SLA performance, and automate reporting workflows.

## Objectives
- Clean and transform large ticketing datasets  
- Conduct SQL and Python-based analysis  
- Build Power BI dashboards for stakeholders  
- Automate reports to reduce manual effort by 30%

## Folder Structure
incident-ticketing-analytics/
├─ data_raw/ # Original dataset from Kaggle
├─ data/ # Cleaned and processed data
├─ notebooks/ # Jupyter notebooks for EDA
├─ scripts/ # Python scripts for ETL
├─ sql/ # SQL schema and queries
├─ powerbi/ # Power BI dashboards
└─ README.md
